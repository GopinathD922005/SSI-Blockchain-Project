📘 About This System

This project implements a Self-Sovereign Identity (SSI) system using blockchain technology. It allows users to control their own identity without relying on a central authority.

⚙️ How It Works

  • User creates a DID (Decentralized Identifier)
  • Issuer generates and signs credentials
  • Credentials are stored securely on blockchain
  • Verifier validates using digital signatures

🧠 Technologies Used

  • Solidity Smart Contracts
  • Ethereum (Ganache Local Blockchain)
  • MetaMask Wallet
  • IPFS (Decentralized Storage)
  • Ethers.js

🌍 Real-World Use Cases

  • 🎓 Digital Student Identity
  • 🏥 Healthcare Records
  • 🏛️ Government ID Systems

🔐 Security Features

  • Blockchain immutability
  • Cryptographic signatures
  • Decentralized storage
