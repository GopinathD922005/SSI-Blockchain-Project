# Blockchain-Based Self-Sovereign Identity (SSI) System for Secure Digital Identity Management

## What's included?

The project includes native support for TypeScript, Hardhat scripts, tasks, and support for Solidity compilation and tests.

# Blockchain-Based Self-Sovereign Identity (SSI)

## 🔐 Overview
This project implements a decentralized identity system using blockchain technology.  
Users can create their own identity (DID), issuers can issue credentials, and verifiers can verify them securely.

---

## 🚀 Features
- Decentralized Identity (DID) Creation
- Credential Issuing with Digital Signature
- Credential Verification
- IPFS File Storage (Pinata)
- MetaMask Wallet Authentication

---

## 🛠️ Technologies Used
- Solidity (Smart Contracts)
- Ganache (Local Blockchain)
- MetaMask (Wallet)
- Ethers.js (Blockchain Interaction)
- IPFS / Pinata (File Storage)
- HTML, CSS, JavaScript (Frontend)

---

## ⚙️ How to Run

1. Start Ganache
2. Deploy contract:
   ```bash
   npx hardhat compile
   node scripts/deploy.js
